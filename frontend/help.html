Help & Guidance

Welcome to the Pahana Edu Billing System help page. Here you can find guidance on how to use the system effectively.

🔑 Login & Authentication

  • Use your username and password to log in.
  • Admins have full access to all modules.
  • System Users have restricted access (e.g., no access to User Management).
  • Inactive sessions will automatically expire after 8 hours.

👥 Customer Details

Use this module to manage customer information. You can:

  • Add new customers
  • Edit existing customer details
  • Search and view customer profiles

🧑‍💻 User Management

Available to Admins only. Functions include:

  • Add new System Users
  • Edit or remove existing System Users
  • Assign roles (Admin or System User)

📦 Manage Items

This module allows you to:

  • Add new products/items
  • Edit item details such as price, stock, and description
  • Remove outdated or inactive items

🧾 Billing

The billing module is used for processing transactions:

  • Create new invoices for customers
  • Apply discounts where applicable
  • Print or export billing details

❓ Need More Help?

If you encounter any issues, please contact your system administrator.
